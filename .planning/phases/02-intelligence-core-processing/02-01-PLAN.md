# Plan 02-01: Model Development

Implement the core unsupervised machine learning models (Autoencoder and Isolation Forest) and the preprocessing pipeline for real-time inference.

## Must Haves
- `Autoencoder` class in PyTorch with encoder/decoder architecture.
- `IsolationForest` wrapper using Scikit-learn.
- `Preprocessor` class for normalization (MinMaxScaler) and basic noise filtering.
- `Trainer` script to fit both models on "normal" synthetic data and save artifacts to `models/`.
- Unit tests to verify that anomalous data produces significantly higher scores than normal data.
